安裝教學

安裝好AnaConda與Python 3.9後

建立 mediapipe 虛擬環境
conda create --name mediapipe python=3.9

進入 mediapipe 虛擬環境
conda activate mediapipe

在虛擬環境中安裝 Jupter
conda install jupyter notebook

安裝 mediapipe
pip install mediapipe

安裝 tensorflow
pip install tensorflow

安裝 opencv
pip install opencv-python==4.5.1.48

最後啟動 Jupter 開發環境
